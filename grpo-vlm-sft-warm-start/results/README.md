# Archived result provenance

The four values in the project README are transcribed from the original coursework materials, not recomputed during this curation.

- **SFT warm-start reward** and **cold-start reward** are reported mean accuracy rewards over 50 logged GRPO steps.
- **Held-out formatting success** reports the share of held-out responses that matched the expected structured answer format.
- **Mean partial score** uses the coursework reward logic, which awards partial credit across structured MILP-answer fields.

Raw training logs, evaluation images, checkpoints, model outputs, and dataset records are excluded because they are large and may require separate redistribution review. The original materials do not establish a public benchmark claim or support a claim that this curated notebook was rerun.
