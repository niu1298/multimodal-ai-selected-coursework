# Archived evaluation context

The complete notebook retains its original result tables, figures, and selected textual outputs. They are historical coursework records and were not regenerated during curation.

The source contains multiple run contexts and written interpretations that do not reconcile to one directly comparable benchmark. This portfolio therefore makes no public claim about a single best fusion method or a consolidated retrieval score.

Raw datasets, per-instance source text, solver logs, local paths, checkpoints, and standalone generated figures are excluded.
